# 🎲 Heads or Tails Java Game

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)

##  Project Overview
This is a simple **console-based Heads or Tails game** built in Java.  
The user plays against the computer, which randomly selects either *heads* or *tails*. The program then compares both choices and determines the winner.

---

##  How It Works
1. The program welcomes the user.
2. The user is prompted to enter **heads** or **tails**.
3. The computer randomly selects heads or tails using Java’s `Random` class.
4. The result is compared:
   - If both match → **Tie**
   - If the player guesses correctly → **Win**
   - Otherwise → **Loss**
5. The user is asked if they want to play again.

---

## Features
-  Random computer choice using `Random`
-  User input validation using `Scanner`
-  Replay option (play again loop)
-  Win / Lose / Tie logic
-  Simple console interface

---

##  Technologies Used
-   Java
- `Scanner` (for user input)
- `Random` (for computer selection)
- Conditional statements (`if / else`)
- Loops (`while`)
